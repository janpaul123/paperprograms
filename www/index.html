Paper Playground

Open all three links:

Camera

Detect or Preview Programs. Create or select playspaces. Print papers. Open program editor.

Projector

MUST BE OPEN TO FOR BOARD TO RUN. See code displayed specifically to the Projector window. Verify programs are detected.

Design Board

Display and change interactive web components instantiated from your Programs.

The Editor is opened from Camera. Currently, the Editor is tied to the Playspace you are creating Programs in.

This tool was adapted from Paper Programs by Jan Paul, which was inspired by the work at Dynamicland.

Find the Paper Playground source code and documentation in the papyrus GitHub repository.

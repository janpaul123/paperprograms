Paper Playground

Interactive Interaction Design

To run the tool, open the three pages below in order:

  1. Camera - Configure your paper detection, change spaces (sets of programs), and access the program editor here!
  2. Projector - Required for code execution. Open before Design Board. Optionally, this window can be used to verify program recognition or display simple canvas graphics or audio.
  3. Design Board - Interactive Design Board. The multimodal components you create in your programs will appear here!

Instructions and tutorials for use are currently under construction. The best documentation at this time is the original paperprograms tutorial.md. Documentation updates coming soon!

This tool was adapted from Paper Programs by Jan Paul Posma, which was inspired by the work at Dynamicland.

Find the Paper Playground source code and documentation in the paper-land GitHub repository.
